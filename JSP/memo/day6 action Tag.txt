action Tag
	서버나 클라이언트에게 어떤 행동을 하도록 명령하는 태그
	페이지와 페이지 사이를 제어하거나 다른 페이지의 실행 결과 내용을
	현재 페이지에 포함시킬때 사용한다. 또한 자바빈즈(자바에서 개발해놓은 객체들을 불러와쓸때 사용)와 같은 다양한 
	기능을 제공하며, 액션 태그는 XML형식의 <jsp:/>를 사용한다.

	1. forward(<jsp:forward />) 
		다른 페이지로의 이동, 페이지 흐름을 제어
	2. include(<jsp:include page=""/>)
		외부 페이지의 내용을 포함하거나 페이지 모듈화
	3. param(<jsp:param />)
		현재 페이지에서 다른 페이지에 정보를 전달한다.
